## Descripción

Este repositorio contiene el desarrollo correspondiente al **Trabajo Práctico: Pilas** de la materia **Algoritmo y Estructura de Datos II**. Se trabaja con los conceptos de **TDA** y **Pilas** (Estructuras dinamicas). Al final se desarrolla una prueba de escritorio, demostrando el desarrollo logico y estructurado de apilar y desapilar.
